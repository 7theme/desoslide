At first, include jQuery and the desoSlide files in your page: 

<link rel="stylesheet" href="css/jquery.desoslide.css" />

<script src="js/jquery.js"></script>
<script src="js/jquery.desoslide.js"></script>

Then, you need to create your thumbnails list with that markup: 

<ul id="my_thumbs">
	<li>
		<a href="img/lake.jpg">
			<img src="img/lake_thumb.jpg" alt="Lake" data-caption="A lake">
		</a>
	</li>
	<li>
		<a href="img/flower.jpg">
			<img src="img/flower_thumb.jpg" alt="Flower" data-caption="A flower">
		</a>
	</li>
	<li>
		<a href="img/butterfly.jpg">
			<img src="img/butterfly_thumb.jpg" alt="Butterfly" data-caption="A butterfly">
		</a>
	</li>
	<li>
		<a href="img/rose.jpg">
			<img src="img/rose_thumb.jpg" alt="Rose" data-caption="A rose">
		</a>
	</li>
</ul>

Specify your images links and the alt attribute. The data-caption attribute is used for the caption.

Call the plugin "desoSlide" on your list of thumbnails by specifying the target where the main image will be generated: 

$(function() {
	$('#my_thumbs').desoSlide({
		mainImage: '#mainImage'
	});
});

Here, the main image will be generated inside #mainImage.

Examples

Displaying the second image at loading. Disabling the caption.

  • Code
$('#ex1_thumbs').desoSlide({
	mainImage: '#ex1_mainImage',
	imgFirst: 1,
	disableCaption: true
});

Displaying the caption by hovering the main image.

  • Code
$('#ex2_thumbs').desoSlide({
	mainImage: '#ex2_mainImage',
	displayCaption: 'hover'
});

Documentation

Find below all the available options.

Options Values Default Description
mainImage Your selector. false The main image container. If you don't specify a valid selector, you'll receive an error.
insertion 'append', 'prepend' or 'replace' 'append How to create the main image, by appending, prepending or replace all the selector's content (overwriting).
imgFirst An integer between 0 and the number of thumbnails (-1). 0 The index of the thumbnail that you want to display first.
disableCaption Boolean. false Disable or not the main image caption. If false, be sure that you have specify the data-caption attribute on thumbnails.
displayCaption 'always' or 'hover' 'always' How to show the main image caption.
displayWarning Boolean. true Show warnings in the developer console. Be sure to put it false in production.
