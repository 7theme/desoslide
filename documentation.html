At first, include jQuery and the desoSlide files in your page: 

<link rel="stylesheet" href="css/jquery.desoslide.css" />

<script src="js/jquery.js"></script>
<script src="js/jquery.desoslide.js"></script>

Then, you need to create your thumbnails list with that markup: 

<ul id="my_thumbs">
	<li>
		<a href="img/lake.jpg">
			<img src="img/lake_thumb.jpg" alt="Lake" data-info="A lake">
		</a>
	</li>
	<li>
		<a href="img/flower.jpg">
			<img src="img/flower_thumb.jpg" alt="Flower" data-info="A flower">
		</a>
	</li>
	<li>
		<a href="img/butterfly.jpg">
			<img src="img/butterfly_thumb.jpg" alt="Butterfly" data-info="A butterfly">
		</a>
	</li>
	<li>
		<a href="img/rose.jpg">
			<img src="img/rose_thumb.jpg" alt="Rose" data-info="A rose">
		</a>
	</li>
</ul>

Specify your images links and the alt attribute. The data-info attribute is used for the caption.

Call the plugin "desoSlide" on your list of thumbnails by specifying the target where the main image will be generated: 

$(function() {
	$('#my_thumbs').desoSlide({
		mainImage: '#mainImage'
	});
});

Here, the main image will be generated inside #mainImage.
