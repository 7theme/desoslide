At first, include jQuery and the desoSlide files in your page: 

<link rel="stylesheet" href="css/jquery.desoslide.css" />

<script src="js/jquery.js"></script>
<script src="js/jquery.desoslide.js"></script>

The CSS file is only needed for the caption styles.

Then, you need to create your thumbnails list with that markup. Don't forget to specify your images links and the alt attribute. 

<ul id="my_thumbs">
	<li>
		<a href="img/bird.jpg">
			<img src="img/bird_thumb.jpg" alt="Bird" data-caption="A bird, click me to open http://chez-syl.fr/" data-href="http://chez-syl.fr/">
		</a>
	</li>
	<li>
		<a href="img/flower.jpg">
			<img src="img/flower_thumb.jpg" alt="Flower" data-caption="A flower" data-caption="A flower, click me to open README.md file" data-href="README.md">
		</a>
	</li>
	<li>
		<a href="img/butterfly.jpg">
			<img src="img/butterfly_thumb.jpg" alt="Butterfly" data-caption="A butterfly" data-caption="A butterfly, click me to open LICENSE.md file" data-href="LICENSE.md">
		</a>
	</li>
	<li>
		<a href="img/rose.jpg">
			<img src="img/rose_thumb.jpg" alt="Rose" data-caption="A rose, you can't click me, got no link">
		</a>
	</li>
</ul>

Explanations:

  • Specify your images links and the alt attribute
  • The data-caption attribute is used for the caption (optional)
  • The data-href attribute contains the link to the page that will be active on the main image (optional)

Call the plugin "desoSlide" on your list of thumbnails by specifying the target where the main image will be generated: 

$(function() {
	$('#my_thumbs').desoSlide({
		mainImage: '#mainImage'
	});
});

Here, the main image will be generated inside #mainImage.

to see how to get that.

Examples

Displaying the second image at loading. Disabling the caption and no link.

  • Code
$('#ex1_thumbs').desoSlide({
	mainImage: '#ex1_mainImage',
	imgFirst: 1,
	enableCaption: false
});

Displaying the caption by hovering the main image. No data-href provided.

  • Code
$('#ex2_thumbs').desoSlide({
	mainImage: '#ex2_mainImage',
	displayCaption: 'hover'
});

Documentation

Find below all the available options.

Options Values Default Description
mainImage Your target container false The main image container. If you don't specify a valid selector, you'll receive an error.
autoLoad Boolean true All thumbnails links (full size images) are auto loaded.
insertion 'append', 'prepend' or 'replace' 'append How to create the main image, by appending, prepending or replace all the selector's content (overwriting).
imgFirst An integer between 0 and the number of thumbnails (-1) 0 The index of the thumbnail that you want to display first.
enableCaption Boolean true Enable or not the main image caption. If true, be sure that you have specify the data-caption attribute on thumbnails.
displayCaption 'always' or 'hover' 'always' How to show the main image caption.
displayWarnings Boolean true Show warnings in the developer console. Be sure to put it false in production.
displayErrors Boolean true Show errors in the developer console. Be sure to put it false in production. If the render isn't ok, enable the errors then debug.

Return value

The returned value by the .desoSlide() call is the jQuery object. However it may return false if:

  • A warning is triggered
  • An error is triggered
