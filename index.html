desoSlide

Lightweight images slider...

... with thumbnails! Feel free to use your own HTML markup!

See documentation & examples

  • Download
  • Version Alpha

Free

This plugin is open source, under the MIT license. You may contribute on the GitHub repository.

Controls

You may control the slider: start/pause the diaporama and so on.

Agile

You are able to use your own CSS. Create your thumbnails by using the HTML template (see "Wizard by step" in the documentation).

Caption

Specify a caption for your images, it's displayed with an overlay.

Multiple

You may use several sliders on a same page.

Tests suite

A Jasmine tests suite is available, see the tests page.
